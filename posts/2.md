## 1-1《编程如此简单—C++程序设计入门》主讲人：朱全民



### 例题：长方形问题

输入长方形的长l和宽w，计算长方形的周长c和面积s

``` C++
#include<iostream>
using namespace std;

int main(){
	int l,w,c,s;
	cin>>l>>w;
	c=2*(l+w);
	s=l*w;
	cout<<"周长="<<c<<endl;
	cout<<"面积="<<s<<endl;
	return 0;
}
```



### 练习题：时间计算

吕布和貂蝉约会，貂蝉迟到了h小时m分钟s秒，请问貂蝉共迟到了多少秒，请编程让计算机回答。

``` C++
#include<iostream>
using namespace std;

int main(){
	int h,m,s;
	cin>>h>>m>>s;
	m=h*60+m;
	s=m*60+s;
	cout<<"迟到了"<<s<<"秒"<<endl;
	return 0;
}
```
